# QA Doctorado 2026 - Equipo X

This repository contains all the work and documentation for the QA Doctorado 2026 project by Team X.

- `setup/` - Environment setup scripts
- `scripts/` - Testing and measurement scripts
- `evidence/` - Weekly evidence collection
- `quality/` - Quality scenarios and glossary
- `risk/` - Risk assessment and test strategy
- `design/` - Test case design and oracle rules
- `ci/` - Continuous integration configuration
- `memos/` - Weekly progress memos
- `reports/` - Unit reports
- `study/` - Research study materials
- `paper/` - Final paper
- `slides/` - Presentation materials
- `peer_review/` - Peer review materials
